# NeuroText
### Proyecto de investigación en nuevas arquitecturas neuronales  
**Tecnologías:** Java · IntelliJ IDEA · Maven

## Descripción del proyecto
NeuroText es un proyecto desarrollado en **Java** utilizando **Maven** como sistema de construcción y gestionado desde **IntelliJ IDEA**. Su objetivo principal es investigar y desarrollar **nuevas arquitecturas neuronales** y **métodos alternativos de entrenamiento** para sistemas de inteligencia artificial.

A diferencia de los modelos convencionales, esta IA propone una estructura en la que **cada neurona representa una letra**, permitiendo un procesamiento lingüístico más granular y simbólico.

El proyecto también busca integrar **mecanismos de aprendizaje por recompensas**, similares a los empleados en redes neuronales para videojuegos, lo que facilita la autocorrección y la optimización dinámica del aprendizaje.

Además, el sistema incorpora **contexto pasado** para mejorar la coherencia y la toma de decisiones, avanzando hacia un modelo de **autoaprendizaje continuo** capaz de adaptarse con la experiencia.

En conjunto, NeuroText aspira a sentar las bases para una nueva generación de inteligencias artificiales con:
- Mayor comprensión contextual  
- Mayor capacidad de adaptación  
- Métodos de entrenamiento más eficientes  
- Integración entre aprendizaje profundo, neurociencia y sistemas simbólicos

## Tecnologías utilizadas
- Java 17 o superior  
- Maven  
- IntelliJ IDEA (JetBrains)

## Objetivos del proyecto
- Experimentar con nuevas formas de representación neuronal.  
- Probar enfoques de aprendizaje por recompensas aplicados al lenguaje.  
- Integrar memoria contextual y autoaprendizaje.  
- Explorar alternativas al entrenamiento clásico de redes neuronales.
