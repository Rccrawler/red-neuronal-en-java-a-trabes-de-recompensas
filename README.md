El proyecto tiene como objetivo explorar y desarrollar nuevas arquitecturas neuronales y métodos de entrenamiento para sistemas de inteligencia artificial. A diferencia de los modelos tradicionales, esta IA propone una estructura única donde cada neurona representa una letra, permitiendo un procesamiento lingüístico más granular y simbólico.  La investigación busca implementar mecanismos de aprendizaje por recompensas similares a los utilizados en redes neuronales para videojuegos, fomentando la autocorrección y la optimización dinámica del aprendizaje. Además, el sistema integrará contexto pasado para mejorar la coherencia y la toma de decisiones, evolucionando hacia un modelo de autoaprendizaje continuo capaz de adaptarse y mejorar con la experiencia.  En conjunto, este proyecto pretende sentar las bases para una nueva generación de inteligencias artificiales con mayor comprensión contextual, capacidad adaptativa y eficiencia en el entrenamiento, fusionando conceptos de neurociencia, aprendizaje profundo y sistemas simbólicos.
